# 网络访问配置指南

## 问题诊断

你遇到的问题是：**SSH连接到Windows电脑时，`localhost` 指向SSH客户端，而不是Windows电脑**

### 三种访问场景

---

## 1️⃣ 本地直接运行（Windows电脑上直接运行Flask）

```bash
python run.py
```

**输出会显示:**
```
🌐 远程访问 (手机/其他设备):
   http://192.168.1.100:5000   # 这是你的实际IP
```

**访问方式:**
- 同一电脑: `http://localhost:5000`
- 其他设备: `http://192.168.1.100:5000`

---

## 2️⃣ SSH远程连接运行（你目前的情况）

```bash
# SSH到Windows电脑
ssh user@192.168.1.100

# 进入项目目录
cd /path/to/english_learning_system

# 启动服务
python run.py
```

**输出示例:**
```
🚀 服务器已启动！

📌 本地访问 (SSH/本机):
   http://localhost:5000

🌐 远程访问 (手机/其他设备):
   http://192.168.1.100:5000   <-- 使用这个
```

### 关键点：
- ❌ **不要** 用 `localhost:5000`（这会连接到你的SSH客户端）
- ✅ **要** 用 `192.168.1.100:5000`（Windows电脑的实际IP）

---

## 3️⃣ 手机/其他设备访问

### 前提条件：
✅ 手机和Windows电脑在**同一局域网**（同一个WiFi）
✅ Windows防火墙允许**5000端口**入站
✅ 知道Windows电脑的IP地址

### 如何获取Windows的IP地址？

#### 方法A：从Flask启动信息
```
🌐 远程访问 (手机/其他设备):
   http://192.168.1.100:5000
```

#### 方法B：Windows命令行查看
```cmd
ipconfig
```

查找 `IPv4 地址` 那一行，通常是 `192.168.x.x` 或 `10.0.x.x`

#### 方法C：路由器后台查看
登录路由器管理界面（通常是 192.168.1.1 或 192.168.0.1）

### 手机访问步骤：

1. **获取Windows IP**
   ```
   比如: 192.168.1.100
   ```

2. **手机浏览器输入**
   ```
   http://192.168.1.100:5000
   ```

3. **应该看到主页**
   ✅ 主仪表板加载
   ✅ 可以添加单词
   ✅ 可以进行复习

---

## 🔧 防火墙配置

如果手机连不上，可能是防火墙阻止了。

### Windows Defender防火墙配置：

1. **打开Windows Defender防火墙**
   - 搜索 "Windows Defender 防火墙"
   - 点击 "允许应用通过防火墙"

2. **添加规则**
   - 点击 "更改设置"
   - 点击 "允许其他应用"
   - 选择 `python.exe`（你的Python安装路径）
   - 点击 "添加"

3. **或者开放5000端口**
   - 进入 "高级设置"
   - 创建 "入站规则"
   - 协议: TCP
   - 本地端口: 5000
   - 操作: 允许

## 🔍 故障排除

### 问题1：手机连不上
```
❌ 连接被拒绝 / timeout
```

**解决:**
1. 检查IP地址是否正确
   ```
   ping 192.168.1.100  # 或你的实际IP
   ```

2. 检查Flask是否正常运行
   ```
   在Windows上应该看到: 🚀 服务器已启动
   ```

3. 检查防火墙
   - Windows Defender 是否允许Python？
   - 5000端口是否开放？

### 问题2：iOS/Android浏览器显示异常
```
❌ 页面布局混乱
```

**解决:**
- 清除浏览器缓存
- 使用最新版浏览器
- 检查响应式CSS是否加载

### 问题3：邮件提醒无法工作
```
❌ 邮件无法发送
```

**原因:** 需要在 `/settings` 页面配置SMTP

**解决:**
1. 在手机上访问: `http://192.168.1.100:5000/settings`
2. 配置SMTP信息（Gmail/Outlook等）
3. 测试邮件配置
4. 启用每日提醒

## 🌐 高级配置

### 使用域名代替IP（可选）

如果你想用域名而不是IP地址：

1. **配置本地DNS（hosts文件）**

   **Windows:** `C:\Windows\System32\drivers\etc\hosts`
   ```
   192.168.1.100    study.local
   ```

   **macOS/Linux:** `/etc/hosts`
   ```
   192.168.1.100    study.local
   ```

2. **手机访问**
   ```
   http://study.local:5000
   ```

### 使用代理/反向代理（高级）

如果你想通过HTTPS或其他配置，可以使用nginx/Apache做反向代理。
